TextBox Directory Structure

  • Live Demo
  • JavaScript
  • HTML
  • CSS

This demo uses a declaration trigger on the key `\`. It reads the current value and sets the declarations to be shown based off of the value. The data shown is hard coded.

Type `C:\`
// directory var directoryStructure = { 'c:': ['inetpub', 'Users', 'Temp', 'Windows'], 'c:\\inetpub': ['wwwroot'], 'c:\\users': ['jdoe', 'jsmith'], 'c:\\temp': ['logs'], 'c:\\windows': ['System', 'System32', 'SYSWOW64'] }; var $editor1 = $('#editor1'); var intellisense1 = new TextBoxIntellisense('editor1'); intellisense1.addDeclarationTrigger({ keyCode: 220 }); intellisense1.onDeclaration(function (item) { var val = $editor1.val().toLowerCase(); if (directoryStructure[val]) { var data = directoryStructure[val] .map(function (item) { return { name: item, glyph: 1 }; }); intellisense1.setDeclarations(data); } });
<input type="text" id="editor1" style="width: 400px;" />
